      Help / About

      Welcome! This app provides Text to Speech (TTS), Speech to Text (SST), and translation features in one place.

      • On Mobile: Select a voice and language from the dropdown menus. Tap “Start Recognition” to begin speech input, or type text for TTS. Use “Speak” to hear your typed text. If you have trouble finding a local voice for an Indian language, we automatically use a fallback TTS service.
      • On Windows: Use Chrome or Edge for best results. Choose a voice from the “Voice” dropdown (some systems have many voices). If the selected voice doesn’t support your chosen language, we’ll fetch audio from Google’s TTS for that language.
      • History: All TTS and SST entries are stored locally. You can search, download (as JSON), or clear them.
      • Settings (gear icon): Toggle Dark Mode, adjust font size, or pick a new font family. Your preferences are saved automatically.

      Enjoy exploring different languages (including Indian languages like Tamil, Telugu, Kannada). If a matching local voice doesn’t exist, we use an online fallback TTS, so you can still hear the text. Also note that translations come from MyMemory’s API, which may have some limitations or errors for certain language pairs.
